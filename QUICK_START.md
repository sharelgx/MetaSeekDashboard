# 快速启动指南

## 🚀 最简单的启动方式

### 第一步：启动前端（独立运行）

```bash
bash start_frontend.sh
```

**前端服务可以独立运行，不依赖后端和数据库！**

### 第二步：访问页面

打开浏览器访问：**http://localhost:5173**

### 第三步：启动项目

页面会显示启动引导界面，你可以：

1. **方式一**：点击"启动项目"按钮（如果后端已运行）
2. **方式二**：在终端执行 `bash start.sh` 启动所有服务

## ✅ 验证

访问 http://localhost:5173，如果能看到页面，说明前端服务正常运行。

## 📝 说明

- 前端服务（Vite）可以独立运行，不依赖后端和数据库
- 即使后端和数据库未启动，前端页面也能正常显示启动引导界面
- 通过启动引导界面，你可以启动后端和数据库服务
- 建议将前端服务设置为系统服务，确保开机自动启动

## 🔧 停止服务

```bash
# 停止所有服务
bash stop.sh

# 仅停止前端
pkill -f vite
```
